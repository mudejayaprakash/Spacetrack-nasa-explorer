Space Data Analytics Dashboard

Interactive insights using Chart.js with dummy data (API-ready for final project)

Note: Currently displaying static dummy data. In the final MVC application, these charts will be populated with real-time data from NASA APIs and your database.

Near-Earth Asteroids Detected (2025)

Monthly count of asteroids passing within 0.05 AU of Earth. Data shows detection patterns throughout the year.

NASA Mission Distribution by Type

Breakdown of active missions by category. Shows the diversity of NASA's current space exploration programs.

Exoplanet Discoveries Over Time

Annual discoveries of confirmed exoplanets and cumulative total from 2016 to 2024. Demonstrates the acceleration of space discovery.

About These Visualizations

Data Sources (Final Implementation):

  • Asteroid Data: Will be fetched from NASA's Near-Earth Object Web Service (NeoWs) API
  • Mission Data: Will be retrieved from NASA's mission database and stored in PostgreSQL
  • Exoplanet Data: Will be sourced from NASA's Exoplanet Archive API

Current Implementation: All charts use dummy data from data.json to demonstrate visualization capabilities. The chart configurations are ready to accept real API data without modification.
